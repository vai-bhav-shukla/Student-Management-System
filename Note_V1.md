# 🎓 Student Management System V1 (Java)

## 📌 Overview

This is my first Java project based on a simple Student Management System. It allows the user to enter the details of three students, calculate their grades, determine the highest marks, and display the complete information.

This project was built to strengthen my understanding of Java fundamentals such as classes, constructors, methods, conditional statements, and user input.

---

## ✨ Features

* Enter details of 3 students.
* Accept student names and marks from the user.
* Validate marks (0–100).
* Automatically calculate grades.
* Find the highest marks among all students.
* Display all student information in a formatted output.

---

## 🛠 Concepts Used

* Java Classes
* Constructors
* Methods
* Scanner Class
* Conditional Statements (`if-else`)
* Ternary Operator
* Variables and Data Types
* User Input Validation

---

## 📂 Project Structure

```
Student.java
```

The project contains a single Java class that performs all operations.

---

## ▶️ How to Run

1. Clone or download the repository.
2. Open the project in VS Code or any Java IDE.
3. Compile the program.
4. Run the `Student` class.
5. Enter the required student details when prompted.
