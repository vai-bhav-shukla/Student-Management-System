# 🎓 Student Management System V2 (Java)

## 📌 Overview

Student Management System V2 is an improved version of my first Java project. In this version, I focused on applying Object-Oriented Programming (OOP) concepts to make the program cleaner, more organized, and easier to maintain.

The project demonstrates how encapsulation and object-oriented design can improve the structure of a Java application.

---

## ✨ Features

* Add student details.
* Store student information using objects.
* Display complete student information.
* Calculate grades based on marks.
* Validate user input.
* Cleaner and more organized code structure compared to Version 1.

## ▶️ How to Run

1. Clone or download the repository.
2. Open the project in VS Code or any Java IDE.
3. Compile the Java files.
4. Run the main class.
5. Enter the required student information.
